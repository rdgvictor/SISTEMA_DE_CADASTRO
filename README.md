# Sistema de Cadastro e Avaliação de Filmes – Go

Projeto desenvolvido em Go (Golang) com o objetivo de praticar lógica de programação, entrada de dados via terminal, uso de funções e laços de repetição.

A aplicação simula um sistema simples de cadastro de filmes e cálculo da média de avaliações informadas pelo usuário.

## Funcionalidades

Exibição de mensagem de boas-vindas

## Cadastro de um filme contendo:

Nome
Ano de lançamento
Preço

## Registro de múltiplas avaliações para o filme

Cálculo da média das notas utilizando laço de repetição (for)

Interação via terminal

## Conceitos Aplicados

Linguagem Go (Golang)

Funções

Entrada de dados pelo terminal

Laços de repetição

Tipos primitivos (string, int, float64)

Organização básica de código

## Como executar o projeto

### Certifique-se de ter o Go instalado
https://go.dev/dl/

### Clone o repositório:

git clone https://github.com/rdgvictor/SISTEMA_DE_CADASTRO


### Acesse a pasta do projeto:

cd nome-do-projeto


### Execute o programa:

go run main.go


## Exemplo de uso

O sistema solicita os dados do filme

Em seguida, pergunta quantas avaliações serão feitas

O usuário digita as notas

O programa calcula e exibe a média final
